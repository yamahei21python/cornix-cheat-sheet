# Cornix Layout チートシート 要件定義

## 1. 概要

Vial形式のキーボードレイアウトファイル `cornix0531.vil` を、
A4横印刷用チートシートとして可視化する。

## 2. 入力データ

- **ファイル**: `cornix0531.vil` (Vial V1形式)
- **キーボード**: Cornix (カスタムCorne系、左右分割)
- **物理構成**: 左右3行×6列 + サムクラスタ3キーずつ (計42キー相当)
- **実レイヤー**: 3層 (Layer 0=Base, Layer 1=Numpad, Layer 2=Fn)
  - Layer 3-9: 全透過(KC_NO)のためチートシート非表示

### 2.1 レイヤー別キー数

| Layer | アクティブ | 透過(KC_NO) | ギャップ(-1) | 計 |
|-------|-----------|-------------|-------------|----|
| 0     | 40        | 10          | 6           | 56 |
| 1     | 23        | 27          | 6           | 56 |
| 2     | 13        | 37          | 6           | 56 |

### 2.2 エンコーダー (2個)

| エンコーダ | Layer 0/1 | Layer 2+ |
|-----------|-----------|----------|
| Encoder 0 | Brt- / Brt+ | ←Layer1以降も同じ |
| Encoder 1 | Vol- / Vol+ / WhUp / WhDn | 全Layer共通 |

→ チートシート下部に補足表示（任意）

## 3. 出力形式

- **ファイル**: `index.html` (単一ファイル、外部依存ゼロ)
- **用紙設定**: A4横 (`@page { size: A4 landscape }`)
- **印刷対応**: `print-color-adjust: exact`、網掛けパターンで白黒印刷可読

## 4. レイアウト構成

```
┌──────────────────────────────────────────────────────────────┐
│  ヘッダー: Cornix Layout                                     │
├────────────────┬────────────────┬────────────────────────────┤
│                │                │                            │
│   Layer 0      │   Layer 1      │   Layer 2                  │
│   BASE         │   NUMPAD       │   FN                       │
│                │                │                            │
│   ┌──キー──┐   │   ┌──キー──┐   │   ┌──キー──┐              │
│   │左右反転│   │   │左右反転│   │   │左右反転│              │
│   └────────┘   │   └────────┘   │   └────────┘              │
│                │                │                            │
```

### 4.1 3層横並び

- A4横を3等分 (各レイヤー約30%)
- 透過キー階層: Layer 0がBase、Layer 1/2の透過キーは枠線のみ

### 4.2 キー物理配置

Corne系カラムスタッガをCSS Gridで再現。

**列スタッガオフセット** (単位: grid単位):

| 列 (inner←→outer) | Yオフセット |
|-------------------|-----------|
| col 0 (outer)     | 0         |
| col 1             | -1        |
| col 2             | -2        |
| col 3             | -2        |
| col 4             | -1        |
| col 5 (inner)     | 1         |

**サムクラスタ**: 3行目の下に配置、inner側2キー + 外側1キー

## 5. 左右対称反転

VILの行列データをそのまま使うと、左手と右手の列方向が不自然。
右手側は列順を反転して表示し、左右対称にする。

### 反転ルール

**左手** (matrix rows 0-2, cols 0→5):

| col0(外) | col1 | col2 | col3 | col4 | col5(内) |
|---------|------|------|------|------|---------|
| 変更なし、そのまま表示 |

**右手** (matrix rows 4-6, cols 0→5 → 表示は col5→0 に反転):

```
VILデータ:  col0(外)  col1  col2  col3  col4  col5(内)
表示:       col5(内)  col4  col3  col2  col1  col0(外)
```

**サムクラスタ**: 同様に反転

```
左手サム:  BTN1(外)  -  -  TD14(内1)  -  Space(内2)
右手サム:  Mute(外)  -  -  Enter(内1)  BSpc(内2)
```

### 6.2 TDラベル

キーキャップ上: メイン動作を表示、小さく `TD(n)` マーク

```
┌──────────┐
│  A       │  ← Tap動作を大きく表示
│  TD0     │  ← 右下に小さく番号
└──────────┘
```

### 6.3 コンボマーク

コンボ参加キー: 右上に小さく `†` マーク
コンボ結果キー: 通常表示

### 6.4 透過キー (KC_NO)

```
┌──────────┐
│          │  ← 2px灰色枠線のみ (塗りつぶしなし)
│          │
└──────────┘
```

### 6.5 ギャップ (-1)

完全非表示 (隙間として扱う)

## 9. VIL更新連携 (vil2sheet.py)

Pythonスクリプト分離:

```
vil2sheet.py cornix0531.vil       # HTML生成
vil2sheet.py cornix0531.vil -o layout.html  # 出力先指定
vil2sheet.py --watch               # ファイル監視モード
```

- 依存: Python 3 + 標準ライブラリのみ
- VILファイル読み込み → 全データパース → HTMLテンプレート埋め込み
- 変更検知 (`--watch`): `inotify`/`watchdog`非依存、ポーリング or 手動

## 10. ファイル構成

```
viral/
├── cornix0531.vil       # 入力: Vialレイアウトデータ
├── vil2sheet.py          # スクリプト: VIL→HTML変換
├── index.html            # 出力: チートシート (スクリプト生成)
└── REQUIREMENTS.md       # 本要件定義書
```

## 11. 非機能要件

- **ブラウザ互換**: Chrome/Firefox/Safari最新版
- **印刷精度**: 各キーがA4印刷で実寸約6-7mm、ラベルが判読可能
- **ファイルサイズ**: index.html 50KB以内
- **生成速度**: スクリプト実行1秒未満
- **白黒印刷**: 網掛けパターンのみで区別可能
