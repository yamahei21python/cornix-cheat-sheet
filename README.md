# vil2sheet.py - Vial キーマップ・チートシート生成ツール

`vil2sheet.py` は、Vial 形式のキーボード設定ファイル（`.vil`）または接続された Vial 対応キーボード（HID）からキーマップ情報を読み取り、印刷・閲覧に最適化された美しい **HTML形式のレイアウトチートシート (`index.html`)** を自動生成するツールです。

特に **Cornix** や **Corne** などのカラムスタッガ（縦ずれ）および左右分割型の30〜40%ミニマルキーボード向けに最適化して設計されています。

---

## 主な機能

- **カラムスタッガ・物理レイアウトの再現**
  左右対称のスタッガ（指の長さに合わせたキーの縦のずれ）や親指クラスタの配置を、CSS を用いて美しく画面上に再現します。
- **マルチレイヤー表示**
  主要な3つのレイヤー（Layer 0: BASE, Layer 1: NUMPAD, Layer 2: FN）を横並びで可視化。透過キー（`KC_TRNS`）は自動的に枠線のみで表現され、上位レイヤーでのキーの役割が直感的にわかります。
- **Tap Dance / Combo / Macro の自動パース**
  キーマップに設定されている Tap Dance（タップ・ホールド切り替えなど）や、同時押し（Combo）、マクロ（Macro）の設定を自動検出し、チートシート下部に詳細テーブルとして自動生成します。
- **2つの入力モード**
  1. **`.vil` ファイルから生成**: Vial アプリからエクスポートした設定ファイルを直接読み込みます。
  2. **キーボードから直接取得**: キーボードを USB 接続した状態で実行し、リアルタイムのメモリから直接レイアウトを抽出します。
- **印刷への最適化**
  用紙サイズ「A4横（A4 Landscape）」での印刷に最適化されたデザインになっており、白黒印刷でもキーカテゴリが判別しやすいように網掛けパターンが設定されています。

## 動作環境・前提条件

- **Python 3.x** （標準ライブラリのみで動作します）
- **hidapi** （※キーボードから直接設定を読み込む機能を利用する場合のみ必要）

### `hidapi` のインストール（直接読み込み機能を使う場合のみ）
```bash
pip install hidapi
```

---

## 使い方

プロジェクトのルートディレクトリで実行します。

### 1. `.vil` ファイルからチートシートを生成する（基本）
Vial アプリ等からエクスポートした `.vil` ファイルを引数に指定して実行します。

```bash
python3 vil2sheet.py cornix0606.vil
```
*実行に成功すると、同ディレクトリに `index.html` が生成されます。*

### 2. 接続されているキーボードから直接読み込んで生成する
キーボードが PC に USB 接続されている場合、ファイルを使わずに現在のキーボード設定から直接チートシートを生成できます。同時に、読み込んだ設定のバックアップファイル（例: `cornix0606.vil`）も自動生成されます。

```bash
python3 vil2sheet.py --from-kb
```

### 3. 出力ファイル名を指定する
デフォルトでは `index.html` として出力されますが、`-o` オプションを使用することで任意のファイル名で出力できます。

```bash
python3 vil2sheet.py cornix0606.vil -o my_layout.html
```

## 印刷時の推奨設定

ブラウザで生成された `index.html` を開き、印刷（PDF出力）する際は以下の設定を推奨します。

- **送信先**: `PDFに保存` または 任意のプリンター
- **用紙サイズ**: `A4`
- **方向**: `横`（Landscape）
- **余白**: `なし` または `最小`
- **オプション**: **`背景のグラフィック` に必ずチェックを入れてください**（キーの色分けや網掛けパターンを表示するために必要です）。
